# Admin Dashboard Screen

## Target
Admin dashboard must use the same mobile card system:

1. Green `#10B981` AppBar
2. Light gray page background
3. White overview card
4. Metric cards using the official palette only
5. Pending order cards
6. Staff availability cards

## Rules
Do not introduce desktop admin panel styling. Keep mobile-first and consistent with customer/staff screens. Use blue for operational/in-progress metrics, amber for pending/attention metrics, purple for support or secondary utility metrics, and slate for text/icons.
